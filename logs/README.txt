Logs are written to this folder automatically.

Main log file:
- samseller-support.log

Rotation settings:
- up to 5 files
- 5 MB per file

Important events included:
- app startup/shutdown and thread start
- Telegram commands from admin
- incoming FunPay messages and filtering reasons
- outgoing messages to FunPay
- delivery retries and errors
- listener crashes and recoveries
