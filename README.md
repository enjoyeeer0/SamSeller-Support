# SamSeller Support

Бот для поддержки FunPay через Telegram.

## Функционал

- Получает новые сообщения с FunPay в Telegram.
- Позволяет отвечать клиенту прямо из Telegram (reply на уведомление).
- Поддерживает список клиентов (`/clients`).
- Поддерживает отправку по коду клиента (`/to C001 ваш текст`).
- Управление доступно только указанному `Admin ID`.

## Как запустить

1. Скачайте архив релиза (`SamSeller-Support-windows-x64.zip`) из раздела `Releases`.
2. Распакуйте архив в любую папку.
3. Запустите `SamSeller-Support.exe`.
4. При первом запуске введите:
	- `Golden Key` FunPay
	- `Telegram Bot Token`
	- `Telegram Admin ID`
5. После этого бот начнет работу.

## Команды в Telegram

- `/help` - список команд
- `/clients` - список последних клиентов
- `/to C001 ваш текст` - отправка клиенту по коду

## Важно

- Файл `secrets.local.json` создается автоматически рядом с программой.
- Не передавайте `secrets.local.json` другим людям.
