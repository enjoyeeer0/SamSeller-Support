# SamSeller Support

Бот для поддержки FunPay через Telegram.

## Функционал

- Получает новые сообщения с FunPay в Telegram.
- Позволяет отвечать клиенту прямо из Telegram (reply на уведомление).
- Поддерживает список клиентов (`/clients`).
- Поддерживает отправку по коду клиента (`/to C001 ваш текст`).
- Управление доступно для 1-3 указанных `Admin ID`.
- Ведет файл логов с ротацией в папке `logs`.

## Как запустить

1. Скачайте архив релиза (`SamSeller-Support-windows-x64.zip`) из раздела `Releases`.
2. Распакуйте архив в любую папку.
3. Запустите `SamSeller-Support.exe`.
4. При первом запуске введите:
	- `Golden Key` FunPay
	- `Telegram Bot Token`
	- `1-3 Telegram Admin ID` (через запятую)
5. После этого бот начнет работу.

Можно также задать переменные окружения:
- `FUNPAY_GOLDEN_KEY`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_ADMIN_IDS` (например: `123456789,987654321`)

Для обратной совместимости поддерживается старый ключ `TELEGRAM_ADMIN_ID`.

## Команды в Telegram

- `/help` - список команд
- `/clients` - клиенты за последние 24 часа
- `/unanswered` - клиенты без ответа за 24 часа
- `/history C001 [N]` - история клиента за последние 24 часа
- `/dialog C001` - включить режим диалога с клиентом
- `/dialog off` - выключить режим диалога
- `/to C001 ваш текст` - отправка клиенту по коду

Также доступны кнопки и inline-действия в уведомлениях, чтобы не вводить команды вручную каждый раз.

## Важно

- Файл `secrets.local.json` создается автоматически рядом с программой.
- Не передавайте `secrets.local.json` другим людям.
- Логи пишутся в `logs/samseller-support.log`.
- В релизном архиве папка `logs` уже присутствует.

## Что логируется

- запуск программы и потоков;
- команды Telegram от администратора;
- входящие события FunPay и причины фильтрации;
- исходящие отправки в FunPay и подтверждения;
- ошибки, ретраи и восстановление listener.
